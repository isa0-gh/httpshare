# Keyboard Shortcuts Guide

httpshare includes comprehensive keyboard shortcuts to speed up your workflow.

## File Operations

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl/Cmd + U` | Upload | Open upload dialog |
| `Ctrl/Cmd + N` | New Folder | Create new folder dialog |
| `Ctrl/Cmd + A` | Select All | Select all files in current directory |
| `Ctrl/Cmd + C` | Copy | Copy selected files to clipboard |
| `Ctrl/Cmd + V` | Paste | Paste copied files to current directory |

## Navigation

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Escape` | Close | Close any open modal/dialog |
| `Arrow Left` | Previous | Previous image in gallery mode |
| `Arrow Right` | Next | Next image in gallery mode |

## Gallery Mode

When viewing images in gallery mode:

| Shortcut | Action |
|----------|--------|
| `←` | Previous image |
| `→` | Next image |
| `Escape` | Close gallery |

## Tips

### Multi-Select Workflow

1. Click checkboxes or use `Ctrl/Cmd + A` to select files
2. Use bulk actions button to perform operations on selected files
3. Or use `Ctrl/Cmd + C` to copy and `Ctrl/Cmd + V` to paste

### Quick Upload

1. Press `Ctrl/Cmd + U` to open upload dialog
2. Or simply drag and drop files anywhere on the page

### Fast Navigation

1. Use search box to quickly find files
2. Click on folders to navigate
3. Use "⬆️ ../" link to go up one directory

## Browser Compatibility

All keyboard shortcuts work on:
- Chrome/Edge (Windows, Mac, Linux)
- Firefox (Windows, Mac, Linux)
- Safari (Mac)

Note: On Mac, use `Cmd` instead of `Ctrl`
